All charts symbols (called glyphs) are defined in this package, examples include legends and rotated text.

This package provides implementations for rendering glyphs in charts.

There are three interfaces in this package, at the time of this writing. These interfaces provide the abstraction needed for rendering glyphs.

  1. IGlyph provides the method contract for the glyphs in a two dimensional coordinate system.

    The coordinate nomenclature is from standard geometry: (x, y).

    Most glyph implement the IGlyph interface by extending SimpleGlyph or TextGlyph, which represent a simple filled rectangle on the chart, or multi-lined text.

  2. The ITextGlyph interface extends the IGlyph interface to provide the extra functionality needed to manage textual content.
  3. The IGlyphLayoutManager interface is implemented by glyphs that manage the layout of more than one glyph. The interface provides a contract of methods that need to be implemented to manage the layout of several glyphs.

All charts get rendered by using three basic data organizing concepts in this package.

This package provides classes for generating and managing the data in charts. The objects Value, Serie and Group provide methods to organize all data represented in a chart.

A Value represents a point on a chart, in a two dimensional coordinate system. The coordinate nomenclature is from standard geometry: (x, y). Most chart types can be described by a series of two dimensional values. Other attributes that effect the rendering of a value can also be given; for example a textual label or the value's shape.

Multiple values form a Serie. A Serie represents a number of value objects arranged one after the other in succession; for example a line in a chart, or stacked bars in a bar chart. Several series form a Group.
